Talf Lang

Practical, Functional, Logical

Language Goals
  • First class support for both functional programming and logic programming
  • Be Pacman-complete: it should be useful in writing real-world software
  • Be friendly: no monads, no high level concepts, in the spirit of Clojure
  • Embrace the power of dynamic languages
